USCIS Documents

Many USCIS Forms have been imported so that agencies are able to download. If the feature is enabled a section on the applicant view will show which PDFs can be downloaded with pre-populated fields like Applicant Name, Address history, Petitioner, Family Members. As well as agency information for interpreter and preparer.

This table shows a download button, the USCIS document Id and a description.

With the search bar, the user can search by any field in this table.

By clicking a download button, the system will process and auto completed fields based on the information for the applicant or pre-configured default values, and display it as an embedded PDF. This embedded PDF can be edited and saved, so it can be completed later.

Note: the save button will be enabled after a change is made. The shortcut Ctrl+S also works for saving.

Once you are ready to print or download, click the Three dot menu on the top right and select print or download.
