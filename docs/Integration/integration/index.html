Integration

ISM can be integrated with any website that can publish information using webhooks. This can be very useful if the agency have a form on your personal or business web application where applicants can fill in their information, and make payments directly. Once the applicant has filed in the information, and saved, the agency can now easily print any USCIS document, modify the data or correct any mistakes.

However, integrations are not only limited to forms. An integration can be used to run any process or even obtain data from ISM, this can be configured for each agency.

How it works#

The integration works with webhooks. This is a way to connect one system to another using a the popular format called JSON (JavasScript Object Notation). System "A" sends information to System "B"'s webhook. Once the information is in System "B", processing can start.
