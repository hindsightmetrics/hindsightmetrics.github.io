Features#

There are four main features accessible through the web. Each feature works independently so agencies can focus on their needs.

Auto Fill#

Auto Fill is focused on accurately completing the USCIS forms. Once an applicant is registered and their information is entered into ISM, the agency will be able to print the forms to PDF with many fields already completed. These fields include, but are not limited to:

  • Applicant Name
  • Applicant Address and Address History
  • Family Members, and petitioner information if necessary
  • Interpreter and Preparer information based on Agency information
  • Fields where Yes or No are common answers (for example Part 8 of I-485)

An agency also has the option of uploading their own documents (Word or PDF) to be used by Auto Fill.

Below is a list of implemented USCIS documents:

  • I-130 PDF - Petition for Relative
  • I-130A PDF - Supplemental Information for Spouse Beneficiary
  • I-131 PDF - Application for Travel Document
  • I-140 PDF - Immigration Petition for Alien Workers
  • I-485 PDF - Application to Register Permanent Residence or Adjust Status
  • I-601A PDF - Application for Provisional Unlawful Presence Waiver
  • I-765 PDF - Application for Employment Authorization
  • I-821D PDF - Consideration of Deferred Action for Childhood Arrivals
  • I-824 PDF - Application for Action on an Approved Application or Petition
  • I-864 PDF - Affidavit of Support Under Section 213A of the INA
  • N-400 PDF - Application for Naturalization

Case Dashboard#

Case Dashboard is a system that tracks any applicant case automatically, using the official USCIS page. Since each applicant has many cases, tracking all of the manually can be a hassle, this is why Case Tracker presents all the cases and their statuses in a single dashboard.

The Case dashboard is automatically updated to show the most up to date status for each of the applicant‚Äôs cases. It is sorted and color coded by latest update time and makes is easy to quickly see which cases need action

Notify#

Notify allows agencies to schedule WhatsApp, SMS or email notifications, based on key milestones or case updates

Automated Call#

Similar to Notify, Automated Call can be configured to make a call based on the agency‚Äôs custom preferences . Examples include:

  • Automatic calls to the Los Angeles immigration court to request court date information based on applicant‚Äôs A-Numbers
  • Automatic calls to applicants whenever a their cases were updated
