Applicant

This is the main view for an applicant. This view has four sections: Applicant Information, Documents, Case and Court Date Check. Below is a description of each section. =>

Applicant Information Area#

Here we can see the information corresponding to the applicant. This form is divided into sections containing name and identity, contact information, main address and ISM Settings:

The applicant details are divided into the following sections. Any field marked with a "*" means that it is required.

Applicant Information#

These fields capture personal details for the applicant:

  1. First Name *
  2. Middle Name
  3. Last Name *
  4. Gender *
  5. Date of Birth *
  6. Nationality

Contact Information#

  1. Email
  2. Phone Number *

Primary Address#

  1. Street Number and Street Name *
  2. Unit Type and Unit Number
  3. City *
  4. State and Zip Code
  5. Province and Postal Code
  6. Country *

Note: Since State and Zip Code are using only for American addresses, Province and Postal Code fields are disabled when Country is set to "USA" or "United States of America". If the country is any other value, State and Zip Code are disabled and Province and Postal Code are enabled. When either field is enabled it is required.

Family Members and Address History will bring up another view More Applicant Details

Notes and Settings#

  1. Notes. In this field the agency can add internal notes or remarks regarding the applicant
  2. Active. Use this toggle if the applicant is no longer active with the agency.
  3. [Case Tracker] "SMS Updates". Text messages will be sent to the applicant whenever there is an update to a case status.
  4. [Automated Call] "Court Date Check". ISM will trigger automated calls to Los Angeles Court to obtain information on any upcoming court dates. Any required fields are validate, and an error message will show when they are empty:

Any validation error will show at the bottom of the form like in the below screenshot

Documents#

The documents area shows for agencies that have the "Auto Fill" feature enabled. This section allows the download of any USCIS document with default information already completed.

This feature also allows for an agency to upload any files and auto fill those too.

  1. To download a document, click on the arrow button.
  2. This table has more then one page, navigate to other pages here.
  3. There is a quick search here.

Cases#

This area shows for agencies that have "Case Tracker" feature enabled.

Here, an agency can link Case Numbers to an applicant, and have ISM automatically check for updates.

On the case table, double clicking allows the Case Id and Case Type to be edited.

  1. Add a new case to the applicant. When this button is clicked, a pop-up will display to input Case Number and Case Id

  1. Manually get the latest status
  2. Get the status for all cases
