Applicants

Selecting "Applicants" on the left sidebar will bring up a view of a list of applicants that the agency is working on and monitoring.

Below is a description of the four actions that a user can take on this view.

Actions#

Create a new applicant#

With the button named "New Applicant" you can create a new applicant. Clicking this button will bring up the Applicant View.

Navigate to an existing applicant#

To view the information for an existing applicant, you can click on any of the Ids shown on the list. By clicking this link the Applicant View will load with that applicant's data.

Search of an applicant by different fields#

A quick search feature is shown on the top right of the applicant table. You can use this search to look up an applicant by various fields like: First Name, Last Name, Phone Number and Email.

Show inactive applicants#

Once an agency has completed all the work needed for an applicant, the applicant can hidden from the view. This helps the agency easily view only active records. To view inactive applicants you can click the link on the top right.
