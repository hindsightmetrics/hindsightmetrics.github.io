Release Notes

2021#

December 3rd Release#

  • Added Last Edit Date to applicant

  • Added Last Download Date to document download

October 7th Release#

  • Added Notes Fields. Use this field to add notes, reminders or remarks regarding the applicant.

September 26th Release#

  • Revamped applicant information and address forms.

September 24th Release#

  • Updated font to Open Sans.

September 17th Release#

  • Added Submenu with links to More Resources and Release Notes, and Contact Us and Submit Request email links

September 16th Release#

  • Agency View Updates and bug fix. Fixed a bug in which the phone number for an agency was not updating. Added description to the different features.

August 29th Release#

  • Case Tracker UI Updates. Minor updates to Case Tracker page.

August 23rd Release#

  • Added More Resources page. The goal for this page is to have links or references to resources that can help with application times or information that can be useful for an agency.

August 16th Release#

  • Include Address History in Forms: N-400, I-821D. Both forms now include the address history when downloading. This is accomplished by saving an applicants address history under Family Member and Address History

July 31st Release#

  • Applicant Address History. Added forms and fields to allow saving of an applicants previous addresses. For a future release, this information will be included in forms that expect this.

May 12th Release#

  • Security Improvements. User Login and API is done using Auth0, changes to both user login and API calls was improved and updated. Performance was improved by changes "Under-the-hood" in the API Server.

April 24th Release#

  • Parent information on Form I-130A. If the applicant has registered parents, they will now pre-populate when downloading the form.

  • Family Member Validation. Adding a family member now validates for fields and for duplicate members. Sorry, we can't allow more than one spouse :(

  • Performance and Stability. The system is now more reliable with under-the-hood improvements.

April 20th Release#

  • Implemented validation for Applicant Information. The applicant form now validates for valid values in names, gender, date of birth, and phone number.

  • Implemented validation for Applicant Address. Address is now validating, including only fill State and Zip Code for US Addresses or only fill Province and Postal Code for International addresses.

April 13th Release#

  • Implemented I-821D - Consideration of Deferred Action for Childhood Arrivals. Form I-821D is now available for auto fill.

  • Added fields for international address. Postal Code and Province fields were added to the Address portion of the applicant's, family member and agency information.

April 9th Release#

  • Improved N-400 Auto fill. N-400 now auto fills the A-Number in the header of all pages, as well as includes a few dates of signature.

April 4th Release#

  • Applicant Family Members. Family member can now be added and deleted (editing an existing member is scheduled for a future release). Any family member can be set as the petitioner which is used to fill out Form I-130.

  • Petitioner Information on Form I-130. The family member set as the petitioner will be used.
