Release Notes

Description#

This view contains information regarding the agency and features that the agency has subscribed to.

Contact Information and Address#

On the left part of the screen is where the Agency information is entered. This information is used for Auto Fill, for the Interpreter and Preparer fields. A future deployment will add the option of having different addresses and contact information for interpreter and preparer

Features#

On the right, the features table contains the list of available features and which ones are active for the agency. Since the features affect the contract between the agency and Hindsight Metrics, these are no editable. With any change a new contract has to be signed.
